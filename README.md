# CoachPulse Web MVP

A browser-based MVP for CoachPulse that can run on iPhone Safari without needing a Mac or Xcode.

## What it includes
- Landing page
- Email/password auth UI scaffold
- Dashboard
- Workout check-in
- AI coach chat UI scaffold
- Firebase setup file
- Vercel/Netlify deployment guide

## Run locally
```bash
npm install
npm run dev
```

## Deploy
Use Vercel or Netlify. See `docs/DEPLOYMENT.md`.
