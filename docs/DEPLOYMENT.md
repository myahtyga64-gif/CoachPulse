# Deploy CoachPulse Web MVP

## Option 1: Vercel
1. Create a Vercel account.
2. Upload this project to GitHub.
3. Import the GitHub repo into Vercel.
4. Framework preset: Vite.
5. Build command:
```bash
npm run build
```
6. Output folder:
```bash
dist
```

## Option 2: Netlify
1. Create a Netlify account.
2. Upload this folder or connect GitHub.
3. Build command:
```bash
npm run build
```
4. Publish directory:
```bash
dist
```

## Option 3: Run from iPhone using Replit
1. Create a Replit account.
2. Create a new React/Vite project.
3. Upload these files.
4. Run:
```bash
npm install
npm run dev
```
